# amv Geolocation UI
amv Geolocation UI is a frontend for amv Geolocation.

- grunt
- npm
- jshint
- angularjs
- Materialize http://materializecss.com
- leaflet

## Build & development
```bash
npm install
grunt
```

Run `grunt` for building and `grunt serve` for preview.

## Testing

Running `grunt test` will run the unit tests with karma.
